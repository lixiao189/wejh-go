# Configs
In this project we use [viper](https://github.com/spf13/viper) to manage our configs

You can find the config.yaml in /config and config it before run

```yaml

database:
  name: 
  host: "127.0.0.1"
  port: 22
  username: root
  password:

redis:
  host:
  port:
  db:0
  username:
  password:

weapp:
  appid:
  secret:

encryptKey: ""
```
